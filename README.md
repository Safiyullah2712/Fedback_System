#Feedback Management Webpage

This is a simple feedback management webpage where users can submit feedback and view it in a dashboard.

##Features

- Users can submit feedback through a form
- Feedback is stored using LocalStorage
-  All submitted feedback is displayed on the dashboard page
- Feedback entries can be managed or deleted
- Simple and user-friendly interface

##How to Use

Open the feedback submission page and fill in the required details such as name, email, and feedback message.
Click the submit button to store the feedback.

Open the dashboard page to view all submitted feedback.
The dashboard displays the feedback list and allows basic management like deleting entries.

##Technologies Used

- HTML
- CSS
- JavaScript

##Project Purpose

This project was created to practice form handling, LocalStorage usage, and basic dashboard management using JavaScript.
